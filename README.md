# 暑假闯关计划 🏆

一个纯前端的暑假习惯追踪工具，适合学生每天记录学习、运动、家务完成情况。

> 单页 HTML，开箱即用，所有数据存浏览器本地。

---

## 功能一览

### 📋 今日打卡
三区打卡：**学习区**（语文、数学、英语、练字、阅读、其他）+ **运动区**（羽毛球、跳绳、核心训练、上肢训练、骑车、其他）+ **家务区**（倒垃圾、拖地、买菜、拾碗筷、学做饭、其他）
达标规则：**学习 + 运动 + 家务各完成一项**即算达标。达标时会有撒花动画 🎉

### 📝 三句话反思 + 心情
每天 3 个轮换问题（150 天不重复），输入即自动保存。选心情后触发 AI 每日鼓励。

### ⏰ 时刻表
右侧固定显示每日时间安排，设置页可自由增删改，自动按时间排序。

### 🗺️ 闯关地图
SVG 石头路径，每天一颗。绿色=已达标、黄色=有记录、灰色=无记录、👑=最棒的一天。点击可查看和修改历史记录。

### 📊 统计
- 每项近 7/30 天完成率柱状图
- 每周达标趋势 + ⭐星级评定
- 心情曲线
- 达标率、达标天数、最棒的一天

### 🏅 成就徽章
17 枚徽章分三级，头部实时显示已获得的。所有勋章集齐时撒花庆祝。

### 📆 周报
周日自动弹出本周小结，含达标天数、项目排名、备注摘录。

### 🎓 终极报告
开学前一天自动弹出，总结整个暑假的成长和收获。

### 🤖 AI 助手 "小关"（可选）
配置 API 后获得 AI 功能：
- **每日鼓励**：根据全部历史数据生成个性化鼓励
- **AI 周报**：替换模板，生成有数据有温度的周报
- **AI 终极报告**：总结整个暑假的成长

支持 OpenAI 和 Anthropic 格式的 API。

### ⚙️ 数据管理
- JSON 备份 / 导入
- CSV 导出（Excel 可直接打开）
- 重置数据

---

## 使用方式

直接打开 `summer-quest-planner.html` 即可使用，无需安装任何东西。

### 首次使用
1. 打开页面 → 填写姓名和开学日期
2. 开始打卡

### AI 助手配置（可选）
设置页 → AI 助手设置 → 填入 API URL、Key、模型名

---

## 技术说明

- 纯 HTML / CSS / JavaScript，无外部依赖
- 数据存储在浏览器 localStorage（`squest_config_v1` + `squest_data_v1`）
- 建议定期导出备份，换设备时导入恢复
